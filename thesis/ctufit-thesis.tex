\ctufitabstractCZE{Tato bakalářská práce se zabývá návrhem a implementací webového rozhraní pro správu a ovládání zařízení IoT. Ovládání těchto zařízení lze provádět manuálně, nebo může uživatel naplánovat spuštění seznamu několika za sebou jdoucích kroků, zvaný sekvence. Sekvence může být také spuštěna manuálně. Součástí práce je také návrh a implementace komunikace tohoto webového rozhraní se zařízeními s důrazem na možnost specifikace vlastního zařízení a tedy částečné modifikace samotného komunikačního procesu uživatelem.}
\ctufitabstractENG{This bachelor thesis deals with the design and implementation of a web interface for managing and controlling IoT devices. Control of these devices can be done manually, or the user can schedule the execution of a list of consecutive steps, called a sequence. The sequence can also be launched manually. The thesis also includes the design and implementation of communication between this web interface and IoT devices, with an emphasis on the ability to specify custom devices and thus partially modify the communication process itself by the user.}
\ctufitkeywordsCZE{IoT, webové stránky, komunikace, MQTT, chytrá zařízení, sekvence, plánované úkony, komunikační kanál, Smart Home, ovládání, PHP}
\ctufitkeywordsENG{IoT, web pages, communication, MQTT, smart devices, sequences, scheduled tasks, communication channel, Smart Home, remote, PHP}
